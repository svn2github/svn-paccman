How to participate on PACCMan ?
This page provides the information needed if you want to work on PAccMan.
Subversion
PAccMan is hosted at sourceforge and the source is on a subversion repository. Note that is better to use subversion from within Netbeans.
Download and install Java SE Development Kit (JDK) version 6 or higher provided by Sun. Follow the instruction on sun java website to do this on your platform.
You will need version 6.0 or higher as PAccMan uses features from this specific release (new desktop APIs, JavaDB database and the new GroupLayout)
Download, install and configure Netbeans 6.0
You must download and install the version 6.0 of Netbeans (currently beta 2) from here.
You must customize the option of NetBeans in order to be consistent with PAccMan project (character encoding must be UTF-8, warnings, ...) This page explains how to do this.
