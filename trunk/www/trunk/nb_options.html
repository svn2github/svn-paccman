Customize NetBeans to work on PaccMan
This page explains how to configure NetBeans if you want to work on PAccMan.
Use specific userdir
Although not required, it is better to use a specific 'userdir' when launching Netbeans to work on paccman. This way, you can change the options of netbeans for this project only without breaking other projects you are working on. Here is the content of the script I made on linux to launch netbeans when I want to work on paccman:
