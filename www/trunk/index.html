What is PAccMan ?
PaccMan stands for Personal ACCount Manager. It is a very simple account manager. I use it for my personal needs which are very limited, so you may find that a few things - actually a lot ! - are missing. Anyway, I hope it will grow to match more and more needs. These screenshots will give you an overview of the currently supported features.
It is written in java (version 6 or higher) and does not use any platform dependant library. So it will hopefully run on any platform supporting Java.
To users
PAccMan is not yet packaged. So you will need to download and install a few extra things. If you want to try it anyway, follow the instructions on this page.
To developers
PAccMan is still in early stage of development. This page provides the information you need if you want to participate.
